HOW TO CUSTOMIZE SUPABASE EMAILS (SENDER & CONTENT)

---------------------------------------------------------
SMTP PROVIDER SETTINGS (COPY & PASTE)
---------------------------------------------------------
Enter these details in: Supabase Dashboard > Project Settings > Auth > SMTP Settings.

### OPTION A: GMAIL (Most Common)
------------------------------------------------
1. Host: smtp.gmail.com
2. Port: 465
3. User: your-email@gmail.com
4. Password: [YOUR APP PASSWORD] 
   - DO NOT use your normal Gmail password.
   - Go to Google Account (myaccount.google.com).
   - Security > 2-Step Verification (Must be ON).
   - Search "App Passwords" at the top or bottom.
   - Create one named "Supabase". Copy that 16-character code.
5. Sender Email: your-email@gmail.com
6. Sender Name: Your App Name

### OPTION B: OUTLOOK / HOTMAIL
------------------------------------------------
1. Host: smtp.office365.com
2. Port: 587
3. User: your-email@outlook.com
4. Password: your-password (or App Password if 2FA is on)
5. Sender Email: your-email@outlook.com

### OPTION C: RESEND / SENDGRID (Professional)
------------------------------------------------
Use the credentials provided by their dashboard.
- Resend Host: smtp.resend.com
- Port: 465
